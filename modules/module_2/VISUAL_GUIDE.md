- **Minimum Width**: 1200px recommended
- **Sidebar**: Fixed 260px width
- **Filter Sidebar**: 320-400px width, collapsible
- **Content**: Flexible, expands to fill space
- **Cards**: Grid layout, 2-4 columns depending on width

## Animations

- **Hover Effects**: 0.2s ease transition
- **Button Press**: Subtle scale/color change
- **Dialog Open**: Fade in
- **Theme Switch**: Instant color transition
- **Card Hover**: Slight elevation increase
- **Status Changes**: Color fade transition

- **High Contrast**: All text meets WCAG 2.1 AA
- **Keyboard Nav**: Full keyboard support
- **Focus Visible**: Clear focus indicators
- **Screen Reader**: Semantic HTML labels
- **Touch Targets**: Minimum 44x44px
- **Font Scaling**: Respects system font size
